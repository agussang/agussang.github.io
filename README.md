# Serpihan yang Tersisa

> *Sebuah Novel Tentang Keberanian Memilih Diri Sendiri*  
> Oleh **Agus Mendo**

Novel pendek digital yang terinspirasi dari kisah nyata, diperkaya dengan puisi-puisi **Jalaluddin Rumi** dan **Kahlil Gibran**.

## 🚀 Deploy ke GitHub Pages

### Langkah 1 — Buat Repository

1. Buka [github.com/new](https://github.com/new)
2. Nama repo: `serpihan-yang-tersisa` (atau nama lain)
3. Set ke **Public**
4. Klik **Create repository**

### Langkah 2 — Push Project

```bash
cd serpihan-yang-tersisa
git init
git add .
git commit -m "Initial commit: Serpihan yang Tersisa"
git branch -M main
git remote add origin https://github.com/USERNAME/serpihan-yang-tersisa.git
git push -u origin main
```

> Ganti `USERNAME` dengan username GitHub kamu.

### Langkah 3 — Aktifkan GitHub Pages

1. Buka repo di GitHub
2. Klik **Settings** (tab paling kanan)
3. Scroll ke section **Pages** (sidebar kiri)
4. Di bagian **Source**, pilih:
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Klik **Save**

### Langkah 4 — Akses Website

Tunggu 1-2 menit, lalu buka:

```
https://USERNAME.github.io/serpihan-yang-tersisa/
```

Selesai! 🎉
